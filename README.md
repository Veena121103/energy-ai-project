## 📌 Overview
This project predicts household energy consumption using a Machine Learning model.  
It is built as an end-to-end application where users can input electrical parameters and get real-time predictions through a web interface.

---

## 🚀 Features
- Real-time energy consumption prediction
- Interactive UI using Streamlit
- Lightweight ML model (fast and efficient)
- Simple and user-friendly interface
- Displays high/normal energy usage alerts

---

## 🛠 Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

---

## 📊 Input Parameters
The model takes the following inputs:
- Voltage
- Sub Metering 1
- Sub Metering 2
- Sub Metering 3
- Global Reactive Power
- Hour
- Day
- Month

---

## 🎯 Output
- Predicted Energy Consumption value
- Alert:
  - ⚠️ High energy usage
  - ✅ Normal usage

---

## 🧠 Machine Learning Model
- Algorithm: Random Forest Regressor
- Optimized for:
  - Fast predictions
  - Low memory usage
  - Real-world deployment
